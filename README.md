# cli

快速生成工具类 or 库，组件库，uni-app 组件库，hooks 库，cli 脚手工具等结构，仅供个人使用

## 快速开始

```shell
npx @babe0806/cli my-app

? Select the boilerplate type (Use arrow keys)
❯ cli                   # 脚手工具
  rc-components         # react 组件库
  rc-hooks              # react hooks库
  tool                  # 工具类
  uniApp-components     # uni-app 组件库
  utils                 # 工具库
```
